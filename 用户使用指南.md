# 拖延风险评估工具 - 用户使用指南

## 🎯 工具概述

拖延风险评估工具是一个基于机器学习的应用，用于评估用户的拖延风险并提供个性化建议。该工具采用 Flask 后端和 HTML/JS 前端开发，支持数据收集和分析功能。

## 🚀 快速开始

### 方式一：一键启动（推荐）

1. 双击运行 `一键启动工具.bat` 文件
2. 选择启动模式：
   - **1. 本地模式**：仅本地和同一WiFi网络用户可访问
   - **2. 公网模式**：生成公网URL，手机和外部用户均可访问
   - **3. 调试模式**：开发人员使用，包含详细日志
3. 按照提示操作，工具会自动完成依赖安装和服务器启动
4. 根据屏幕显示的URL访问工具

### 方式二：手动启动

1. 打开命令提示符或终端
2. 导航到工具目录：`cd /d D:\procrastination_tool`
3. 运行启动脚本：
   ```bash
   # 本地模式
   python start_app.py
   
   # 公网模式（推荐用于外部用户访问）
   python start_app.py --ngrok
   
   # 自定义端口
   python start_app.py --port 8080
   
   # 调试模式
   python start_app.py --debug
   ```

## 📱 手机用户访问

### 本地网络访问
1. 确保手机和电脑连接同一WiFi网络
2. 在启动工具时获取**局域网访问地址**
3. 在手机浏览器中输入该地址

### 公网访问
1. 选择**公网模式**启动工具
2. 获取生成的**内网穿透URL**
3. 手机用户可直接访问该URL
4. 可将该URL分享给更多用户使用

## 📊 数据收集

### 自动收集的数据
- 用户评估的各项指标（任务厌恶度、结果价值感、自我控制能力）
- 拖延概率预测结果
- 评估时间戳
- 访问来源（如抖音、视频号等）

### 数据存储位置
- 数据保存在 `data/risk_records.db` 文件中
- 采用 SQLite 数据库，可使用 SQLite 管理工具查看

### 数据导出
- 可通过 API 导出脱敏数据用于分析
- 访问 `http://localhost:5001/export-data` 获取导出数据

## 🔧 常见问题

### Q: 工具启动失败怎么办？
A: 检查Python版本是否为3.8或以上，确保网络连接正常，查看错误信息并尝试解决。

### Q: 如何查看收集的数据？
A: 使用SQLite管理工具（如DB Browser for SQLite）打开 `data/risk_records.db` 文件。

### Q: 如何导出数据？
A: 启动工具后，访问 `http://localhost:5001/export-data` 即可获取JSON格式的脱敏数据。

### Q: 公网URL失效怎么办？
A: 重新启动工具，会生成新的公网URL。如需稳定URL，建议配置ngrok认证令牌。
