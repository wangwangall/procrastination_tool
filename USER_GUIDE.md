# 用户指南

## 工具介绍

拖延风险评估工具是一款基于机器学习的应用，通过三个维度评估用户的拖延风险并提供个性化建议。该工具结合了时间决策理论和机器学习技术，能够预测用户的拖延概率并给出针对性的建议。

## 安装与启动

### 安装步骤
1. **创建虚拟环境**：
   ```bash
   conda create -n procrastinate_env python=3.8
   ```

2. **激活环境**：
   ```bash
   conda activate procrastinate_env
   ```

3. **安装依赖**：
   ```bash
   pip install flask sqlite3 scikit-learn pickle-mixin
   ```

4. **生成种子数据**（首次运行时）：
   ```bash
   python seed_data.py
   ```

5. **训练模型**（首次运行时）：
   ```bash
   python train_model.py
   ```

### 启动应用
```bash
python app.py
```

启动成功后，会看到类似以下输出：

## 使用方法

### 访问工具
在浏览器中输入 `http://127.0.0.1:5000` 访问工具页面。

### 评估流程
1. **调整滑动条**：
   - **任务厌恶度**：0-100，值越高表示对任务的厌恶程度越高
   - **结果价值**：0-100，值越高表示任务结果对您的价值越大
   - **自我控制**：0-100，值越高表示您的自我控制能力越强

2. **点击评估按钮**：点击页面上的"评估风险"按钮

3. **查看结果**：
   - 系统会显示您的拖延概率（0-1之间的数值）
   - 系统会根据评估结果给出个性化建议

4. **反馈实际情况**：（可选）
   - 完成任务后，您可以反馈实际是否拖延
   - 这些数据将用于改进模型预测准确性

## 结果解读

### 拖延概率
- **0.0-0.3**：低风险，您不太可能拖延
- **0.3-0.7**：中等风险，您有可能拖延
- **0.7-1.0**：高风险，您很可能拖延

### 建议示例
- **低风险**："您的拖延风险较低，继续保持良好的工作习惯"
- **中等风险**："您的拖延风险中等，建议设定明确的目标和时间计划"
- **高风险**："您的拖延风险较高，建议分解任务并设置明确的时间节点"

## 常见问题

### Q: 工具的预测准确率如何？
A: 基于当前的种子数据，模型训练准确率约为90%。随着数据量的增加，预测准确率会进一步提高。

### Q: 我的数据会被保存吗？
A: 是的，您的评估数据会被匿名保存到本地数据库中，用于模型训练和改进。

### Q: 如果我不想提交数据怎么办？
A: 您可以只使用评估功能，不提交反馈数据。系统仍然会为您提供评估结果和建议。

### Q: 工具需要联网吗？
A: 不需要，工具完全在本地运行，所有数据都存储在本地数据库中。

### Q: 如何更新模型？
A: 当您收集了足够的新数据后，可以运行 `python train_model.py` 重新训练模型。

### Q: 工具支持哪些浏览器？
A: 工具支持所有现代浏览器，包括Chrome、Firefox、Safari和Edge。

## 故障排除

### 常见错误

1. **模型加载失败**
   - 症状：系统提示"模型加载失败，使用时间决策理论" 
   - 解决方法：运行 `python train_model.py` 重新训练模型

2. **数据库连接失败**
   - 症状：系统提示"数据库连接失败" 
   - 解决方法：确保 `data` 目录存在，且具有写入权限

3. **端口被占用**
   - 症状：启动时提示"Address already in use"
   - 解决方法：关闭占用端口的其他程序，或修改 `app.py` 中的端口号

### 性能问题

1. **页面加载缓慢**
   - 解决方法：清除浏览器缓存，或使用Chrome等现代浏览器

2. **评估响应延迟**
   - 解决方法：确保您的计算机性能良好，或关闭其他占用资源的程序
