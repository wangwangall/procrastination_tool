# 手机无法访问服务器的解决方案

## 最常见原因：阿里云安全组未开放端口

### 1. 登录阿里云控制台
- 打开：https://ecs.console.aliyun.com/
- 登录您的阿里云账号

### 2. 进入安全组配置
1. 左侧菜单 → **实例与镜像** → **实例**
2. 找到您的ECS实例（8.148.7.83）
3. 点击实例名称，进入实例详情页
4. 点击 **安全组** 标签
5. 点击 **配置规则** 按钮

### 3. 添加安全组规则
1. 点击 **入方向** → **添加规则**
2. 配置以下规则：
   - **协议类型**: TCP
   - **端口范围**: 80/80
   - **授权对象**: 0.0.0.0/0
   - **描述**: HTTP访问
3. 再添加一条规则：
   - **协议类型**: TCP
   - **端口范围**: 5001/5001
   - **授权对象**: 0.0.0.0/0
   - **描述**: 应用访问
4. 点击 **保存**

### 4. 验证配置
- 规则添加后，等待1-2分钟生效
- 手机浏览器访问：`http://8.148.7.83`

## 其他可能原因

### 1. 手机网络问题
- 检查手机是否使用了 **移动数据** 或 **不同的WiFi**
- 尝试在不同网络环境下访问

### 2. 手机浏览器缓存问题
- 清除手机浏览器缓存
- 或使用隐私模式访问

### 3. 服务器防火墙问题
```bash
# 在服务器上执行以下命令
firewall-cmd --zone=public --add-port=80/tcp --permanent
firewall-cmd --zone=public --add-port=5001/tcp --permanent
firewall-cmd --reload
firewall-cmd --list-ports
```

### 4. 验证服务状态
```bash
# 在服务器上执行以下命令
# 检查Nginx状态
systemctl status nginx

# 检查Gunicorn状态
systemctl status procrastination-tool

# 检查端口监听
netstat -tlnp | grep -E "(80|5001)"
```
