## 📁 GitHub开源准备

### 1. 创建GitHub仓库

1. 登录GitHub，创建新仓库
2. 设置仓库为**Public**（开源）
3. **选择合适的开源协议**（重要！涉及数据收集）
   - 推荐选择：
     - 📝 **MIT License**：最宽松的开源协议，允许几乎任何使用、修改和分发
     - 🛡️ **Apache License 2.0**：包含专利保护条款，更适合涉及数据处理的项目
   - 不推荐：GPL v3/AGPL v3（强copyleft协议，可能限制商业使用）
4. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/procrastination-tool.git
   cd procrastination-tool
   ```

由于本工具涉及用户数据收集（三个维度的评估数据），请务必：

### 5. 添加Docker支持

创建`Dockerfile`：

```dockerfile
FROM python:3.8-slim

WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y --no-install-recommends \
    gcc \
    g++ \
    git \
    && rm -rf /var/lib/apt/lists/*

# 复制依赖文件
COPY requirements.txt .

# 安装Python依赖
RUN pip install --upgrade pip && pip install -r requirements.txt

# 复制项目文件
COPY . .

# 创建数据目录
RUN mkdir -p data/reports

# 暴露端口
EXPOSE 5001

# 启动命令
CMD ["gunicorn", "--workers", "3", "--bind", "0.0.0.0:5001", "app:app"]
```

创建`docker-compose.yml`：

```yaml
version: '3.8'

services:
  app:
    build: .
    ports:
      - "5001:5001"
    volumes:
      - ./data:/app/data
    environment:
      - FLASK_ENV=production
      - SECRET_KEY=your-secret-key-here
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:5001"]
      interval: 30s
      timeout: 10s
      retries: 3
```

### 6. 添加GitHub Actions CI/CD

创建`.github/workflows/deploy.yml`：

```yaml
name: Deploy to Server

on:
  push:
    branches: [ main ]

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Python
      uses: actions/setup-python@v4
      with:
        python-version: '3.8'
    
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install -r requirements.txt
    
    - name: Run tests
      run: |
        # 添加测试命令
        echo "Running tests..."
    
    - name: Build Docker image
      run: docker build -t procrastination-tool .
    
    - name: Deploy to server via SSH
      uses: appleboy/ssh-action@master
      with:
        host: ${{ secrets.SERVER_HOST }}
        username: ${{ secrets.SERVER_USERNAME }}
        key: ${{ secrets.SERVER_SSH_KEY }}
        script: |
          cd /var/www/procrastination-tool
          git pull origin main
          docker-compose down
          docker-compose up -d --build
```

## 📦 部署方案

### 📋 方案1：传统服务器部署（脚本方式）

#### 1. 创建部署脚本

```bash
#!/bin/bash
# 拖延风险评估工具部署脚本
# 适用于 Ubuntu 22.04 LTS

set -e

echo "======================================="
echo "🎯 拖延风险评估工具 - 服务器部署脚本"
echo "======================================="
echo

# 配置
GIT_REPO="https://github.com/your-username/procrastination-tool.git"  # 替换为您的GitHub仓库地址
PROJECT_DIR="/var/www/procrastination-tool"
ENV_FILE="$PROJECT_DIR/.env"

# 更新系统
echo "🔄 更新系统包..."
sudo apt update && sudo apt upgrade -y

# 安装依赖
echo "📦 安装必要依赖..."
sudo apt install -y python3 python3-pip python3-venv git nginx ufw sqlite3 curl certbot python3-certbot-nginx

# 配置防火墙
echo "🔥 配置防火墙..."
sudo ufw allow 22/tcp  # SSH
sudo ufw allow 80/tcp  # HTTP
sudo ufw allow 443/tcp # HTTPS
sudo ufw allow 5001/tcp # 应用端口
sudo ufw enable -y

# 创建项目目录
echo "📁 创建项目目录..."
sudo mkdir -p $PROJECT_DIR
sudo chown -R $USER:$USER $PROJECT_DIR

# 克隆项目
echo "📥 从GitHub克隆项目代码..."
cd $PROJECT_DIR
git clone $GIT_REPO .

# 创建虚拟环境
echo "🌐 创建Python虚拟环境..."
python3 -m venv venv
source venv/bin/activate

# 安装Python依赖
echo "📦 安装Python依赖..."
pip install --upgrade pip
pip install -r requirements.txt

# 安装Gunicorn（生产服务器）
echo "🔧 安装Gunicorn..."
pip install gunicorn

# 创建环境变量文件
echo "⚙️  创建环境变量配置..."
cat > $ENV_FILE << EOL
FLASK_APP=app.py
FLASK_ENV=production
SECRET_KEY=$(openssl rand -hex 16)
DATABASE_URL=data/procrastination_notebook.db
HOST=0.0.0.0
PORT=5001
LOG_LEVEL=INFO
EOL

# 配置Gunicorn服务
echo "⚙️  配置Gunicorn服务..."
sudo tee /etc/systemd/system/procrastination-tool.service > /dev/null << EOL
[Unit]
Description=Gunicorn instance to serve procrastination tool
After=network.target

[Service]
User=$USER
Group=www-data
WorkingDirectory=$PROJECT_DIR
Environment="PATH=$PROJECT_DIR/venv/bin"
Environment="FLASK_ENV=production"
ExecStart=$PROJECT_DIR/venv/bin/gunicorn --workers 3 --bind 0.0.0.0:5001 app:app

[Install]
WantedBy=multi-user.target
EOL

# 配置Nginx反向代理
echo "🌍 配置Nginx反向代理..."
sudo tee /etc/nginx/sites-available/procrastination-tool > /dev/null << EOL
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://localhost:5001;
        proxy_set_header Host host;
        proxy_set_header X-Real-IP remote_addr;
        proxy_set_header X-Forwarded-For proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto scheme;
    }
}
EOL

# 启用Nginx配置
sudo ln -sf /etc/nginx/sites-available/procrastination-tool /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default

# 测试Nginx配置
echo "🧪 测试Nginx配置..."
sudo nginx -t

# 创建数据目录
echo "📊 创建数据目录..."
mkdir -p $PROJECT_DIR/data/reports
chmod -R 755 $PROJECT_DIR/data

# 初始化数据库
echo "🗄️  初始化数据库..."
source $PROJECT_DIR/venv/bin/activate
cd $PROJECT_DIR
python app.py 2>&1 | head -20

# 启动服务
echo "🚀 启动服务..."
sudo systemctl daemon-reload
sudo systemctl start procrastination-tool
sudo systemctl enable procrastination-tool
sudo systemctl restart nginx

# 配置定时备份
echo "💾 配置数据备份..."
sudo tee /etc/cron.daily/procrastination-tool-backup > /dev/null << EOL
#!/bin/bash
PROJECT_DIR="$PROJECT_DIR"
BACKUP_DIR="$PROJECT_DIR/backup"
mkdir -p BACKUP_DIR
date=$(date +%Y%m%d_%H%M%S)
sqlite3 PROJECT_DIR/data/procrastination_notebook.db ".backup '$BACKUP_DIR/risk_records_date.db'"
# 保留最近7天的备份
find BACKUP_DIR -name "*.db" -mtime +7 -delete
EOL

sudo chmod +x /etc/cron.daily/procrastination-tool-backup

# 配置HTTPS
echo "🔒 配置HTTPS（推荐）..."
echo "📝 如需配置HTTPS，请执行以下命令："
echo "   sudo certbot --nginx -d your-domain.com"
echo "   sudo systemctl restart nginx"
echo

echo "======================================="
echo "🎉 部署完成！"
echo "======================================="
echo "🌍 访问地址：http://您的服务器IP 或 http://your-domain.com"
echo "💻 项目目录：$PROJECT_DIR"
echo "📊 数据目录：$PROJECT_DIR/data"
echo "📁 备份目录：$PROJECT_DIR/backup"
echo "🔧 日志查看：sudo journalctl -u procrastination-tool"
echo "======================================="
echo
```

#### 2. 执行部署

```bash
# 赋予脚本执行权限
chmod +x deploy.sh

# 运行部署脚本
./deploy.sh
```
