# 正确配置阿里云轻量应用服务器防火墙

## 您当前的位置
您现在在 **防火墙模板** 的规则列表页面，这是用于创建防火墙模板的，不是用于配置实例防火墙的。

## 正确的路径

### 1. 回到轻量应用服务器列表页
- 点击左上角的返回箭头，一直返回到 **轻量应用服务器列表** 页面
- 或直接访问：https://swas.console.aliyun.com/

### 2. 进入实例详情页
1. 在轻量应用服务器列表中，找到您的服务器实例（IP: 8.148.7.83）
2. 点击实例名称，进入 **实例详情页**

### 3. 进入防火墙配置页面
在实例详情页的左侧菜单中，找到并点击 **防火墙** 选项卡

### 4. 添加防火墙规则
1. 点击 **添加规则** 按钮
2. 配置规则：
   - **应用类型**：HTTP
   - **端口范围**：80/80
   - **授权对象**：0.0.0.0/0
   - **描述**：HTTP访问
3. 点击 **确定**
4. 再添加一条规则：
   - **应用类型**：自定义TCP
   - **端口范围**：5001/5001
   - **授权对象**：0.0.0.0/0
   - **描述**：应用访问
5. 点击 **确定**

## 为什么不能使用模板？
- 防火墙模板需要 **应用到实例** 才能生效
- 您可能没有权限将模板应用到实例
- 直接在实例详情页配置防火墙更简单、更直接
